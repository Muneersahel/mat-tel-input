Country Filtering

Restrict or prioritize which countries appear in the dropdown.

preferredCountries Phone 
[preferredCountries]="['us', 'gb', 'de']"

These countries appear at the top of the dropdown, above a divider.

onlyCountries Phone 
[onlyCountries]="['us', 'ca', 'gb', 'au']"

Only these four countries appear in the dropdown.

API Reference

Inputs

Property Type Default Description
enablePlaceholder boolean false Show example phone number as placeholder
enableSearch boolean false Show search input in country dropdown
format PhoneNumberFormat 'default' Phone formatting: 'default' | 'national' | 'international'
preferredCountries string[] [] ISO-2 codes shown at top of dropdown
onlyCountries string[] [] Restrict dropdown to these ISO-2 codes only
placeholder string '' Custom placeholder text
searchPlaceholder string 'Search country or code' Placeholder for the search input in dropdown
maxLength string | number 15 Maximum input length
resetOnChange boolean false Clear input when country changes
autocomplete 'off' | 'tel' 'off' HTML autocomplete attribute
cssClass string undefined CSS class for the phone input element
required boolean false Mark control as required
disabled boolean false Disable the control

Outputs

Property Type Description
countryChanged EventEmitter<Country> Emits when the selected country changes

Exports

Symbol Type Description
MatTelInput Component The telephone input component (selector: mat-tel-input)
matTelInputValidator ValidatorFn Validates phone numbers; error key: validatePhoneNumber
Country Interface Country object (name, iso2, dialCode, flagClass, ...)
PhoneNumberFormat Type 'default' | 'national' | 'international'
